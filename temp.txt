== Applicability gate ==

Both A and B apply ONLY when the course is "Discovery-driven":

def _is_discovery_driven(candidate, allocator_remaining):
    """True when all of the candidate's unmet buckets are under MCC_DISC*.
    If the course fills ANY non-Discovery unmet bucket, return False —
    no penalty applied."""
    unmet_fills = [
        bid for bid in candidate.get("fills_buckets", [])
        if allocator_remaining.get(bid, {}).get("slots_remaining", 0) > 0
    ]
    if not unmet_fills:
        # Bridge-only or no unmet buckets — check bridge targets
        bridge_targets = candidate.get("bridge_target_buckets", [])
        if not bridge_targets:
            return False
        return all(
            bid.startswith("MCC_DISC") or "::MCC_DISC" in bid
            for bid in bridge_targets
        )
    return all(
        _local_bucket_id(bid).upper().startswith("MCC_DISC")
        or _bucket_parent_map_lookup(bid).startswith("MCC_DISC")
        for bid in unmet_fills
    )

If _is_discovery_driven returns False, both penalties are 0.

            if slots > 0:

Output: integer (e.g. 0 to ~10)
  0 = foundation complete, Discovery competes freely
  5 = half open, Discovery pushed well below foundation picks
  10 = wide open, Discovery nearly invisible in early semesters

Computed once per semester, not per candidate.

== Penalty B: Department affinity ==

def _discovery_affinity_tier(course_code, declared_dept_set):
    """Returns 0, 1, or 2.
    0 = declared-path or business-adjacent (no penalty)
    1 = neutral / general-ed style
    2 = far / off-path starter"""
    dept = course_code.split()[0] if " " in course_code else course_code[:4]
    dept = dept.strip().upper()

    # Tier 0: student's own declared departments
    if dept in declared_dept_set:
        return 0

    # Tier 0: business school departments
    BUSINESS_DEPTS = {
        "ACCO", "BUAN", "ECON", "FIN", "HURE",
        "INSY", "ISCO", "MANA", "MARK", "OSCM", "REAL",
    }
    if dept in BUSINESS_DEPTS:
        return 0

    # Tier 1: common gen-ed / humanities that are normal Discovery picks
    NEUTRAL_DEPTS = {
        "PHIL", "THEO", "ENGL", "HIST", "COMM", "PSYC",
        "SOCI", "POSC", "MATH", "WRIT",
    }
    if dept in NEUTRAL_DEPTS:
        return 1

    # Tier 2: everything else (BIOL, CHNS, PHYS, SPAN, ARAB, etc.)
    return 2

Output: 0, 1, or 2
Only used as tiebreak among Discovery-driven candidates.

== How declared_dept_set is built ==

Extract department prefixes from the student's declared major/track bucket
mappings at runtime. Example: if student declares FIN_MAJOR, their buckets
map to courses with FIN prefix → "FIN" goes in the set. Also include
departments from any selected track's bucket courses.

Built once per recommendation request from existing bucket/course data.

== What this does NOT do ==

- Does not hard-ban any Discovery course
- Does not use standing as a signal
- Does not require per-course manual tagging
- Does not add student-facing preferences or modes
- Does not change eligibility.py at all — purely a ranking change
- Does not affect non-Discovery courses in any way

== Test cases to write ==

1. Freshman with 0 foundation done, declares FIN_MAJOR + MCC_DISC_CB:
   - ECON 1001 (Discovery + fills major bucket) → no penalty (not Discovery-driven)
   - PSYC 1001 (Discovery only) → penalty A = high, penalty B = 1
   - CHNS 1001 (Discovery only) → penalty A = high, penalty B = 2
   - Expected order: ECON 1001 >> PSYC 1001 >> CHNS 1001

2. Junior with foundation nearly done, declares FIN_MAJOR + MCC_DISC_CB:
   - PSYC 1001 (Discovery only) → penalty A = low, penalty B = 1
   - CHNS 1001 (Discovery only) → penalty A = low, penalty B = 2
   - Expected: both rank higher than case 1, PSYC still beats CHNS

3. Student with foundation complete:
   - All Discovery courses → penalty A = 0, only B matters
   - Business-adjacent Discovery beats far-off Discovery

4. Course fills Discovery AND major bucket:
   - _is_discovery_driven returns False
   - No penalty applied at all
